浏览器版本过低，建议您对浏览器进行升级，以便获得更好的浏览体验

为什么要升级浏览器?

您正在使用的是非常落后的浏览器,

为了您的隐私和数据安全,

为了让您得到更好的网上浏览体验,

我们强烈建议您将浏览器升级至以下

1. 存在严重安全漏洞

谷歌、雅虎等其他20多家公司都遭到了黑客的攻击，而黑客所利用的则是IE中的漏洞。

2. 微软停止维护更新

微软在其安全研究和防御博客中指出，为了避免受到攻击的风险，用户最好能够升级IE浏览器。

3. 大量新特性都不支持

IE无法支持新技术，为了照顾它，我们不得不 含泪抹杀掉新创意，新想法，新技术。

4. 太老，太慢了

在这个软硬件飞速更新的时代，IE6的速度和效率极慢极低，在开发人员中已经久负盛名。
